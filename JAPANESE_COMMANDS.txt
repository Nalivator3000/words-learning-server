QUICK SETUP (ONE COMMAND)
================================================================================

  # Windows PowerShell
  $env:ANTHROPIC_API_KEY="your-api-key-here"
  node setup-japanese-complete.js

  # Windows CMD
  set ANTHROPIC_API_KEY=your-api-key-here
  node setup-japanese-complete.js

  # Linux/Mac
  export ANTHROPIC_API_KEY="your-api-key-here"
  node setup-japanese-complete.js

  TIME: 45-90 minutes
  COST: ~$3-5 USD


CHECK STATUS
================================================================================

  node check-japanese-status.js

  Shows:
  - Total words (target: 10,000)
  - Real vs synthetic breakdown
  - Distribution by level
  - Distribution by theme
  - Word sets created
  - What to do next


INDIVIDUAL STEPS
================================================================================

  Step 1: Cleanup synthetic data
  -------------------------------
  node cleanup-japanese-synthetic.js

  Time: < 1 minute
  What: Deletes test data like "はい_123_A1"


  Step 2: Generate vocabulary
  ----------------------------
  node generate-japanese-vocabulary.js

  Time: 30-60 minutes
  What: Creates 10,000 real Japanese words
  Needs: ANTHROPIC_API_KEY


  Step 3: Assign themes
  ----------------------
  node assign-japanese-themes-llm.js

  Time: 20-30 minutes
  What: Categorizes words by theme
  Needs: ANTHROPIC_API_KEY


  Step 4: Create word sets
  -------------------------
  node create-japanese-word-sets.js

  Time: < 1 minute
  What: Creates thematic learning sets


WINDOWS BATCH FILE
================================================================================

  # Set API key first
  set ANTHROPIC_API_KEY=your-key

  # Then double-click or run:
  run-japanese-setup.bat


GET API KEY
================================================================================

  1. Go to: https://console.anthropic.com/
  2. Sign up / Log in
  3. Create API key
  4. Copy and set in environment


VERIFY RESULTS
================================================================================

  After setup, run:
  node check-japanese-status.js

  Expected:
  - Total words: ~10,000
  - Real words: 100%
  - Themed words: 100%
  - Word sets: ~100-150


TROUBLESHOOTING
================================================================================

  Problem: "API key not set"
  Solution: $env:ANTHROPIC_API_KEY="your-key"

  Problem: Rate limit errors
  Solution: Wait, scripts auto-delay

  Problem: Script failed partway
  Solution: Safe to re-run

  Problem: Not enough words
  Solution: Re-run generate script
