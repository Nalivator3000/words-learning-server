# ⚡ Quick Start Guide

Get your Language Learning App up and running in 5 minutes!

## 🚀 1-Minute Local Setup

1. **Download all files** to a folder on your computer
2. **Open `index.html`** in any modern browser (Chrome, Firefox, Safari, Edge)
3. **Done!** The app is ready to use locally

## 🌐 5-Minute Web Deployment

### GitHub Pages (Free)
1. Create new GitHub repository
2. Upload all project files
3. Go to Settings → Pages → Source: main branch
4. Access via `https://yourusername.github.io/repository-name`

### Netlify (Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag & drop your project folder
3. Get instant URL like `https://random-name.netlify.app`

### Vercel (Free)
1. Go to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. Deploy automatically

## 📝 First Steps After Setup

1. **Create Account**: Register with email/password or use Google login
2. **Import Words**: Upload the included `test-words.csv` for demo
3. **Try Modes**: Test all learning modes, especially Survival Mode!
4. **Customize**: Create your language pairs in Settings

## 🔧 Troubleshooting

**App won't load?**
- Use modern browser (Chrome 90+, Firefox 88+)
- Check browser console (F12) for errors
- Ensure all files are in same directory

**Audio not working?**
- Check browser audio permissions
- Try different browser
- Audio works best on HTTPS sites

**Data not saving?**
- Check if IndexedDB is enabled
- Try different browser
- Clear browser cache and retry
