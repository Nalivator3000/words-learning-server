# Hindi Dictionary Contains Placeholders

## Issue Summary
User 87 (test account test_hi_en) reports that their Hindi dictionary shows placeholder text instead of actual words.

## Investigation Results

### Account Details
- **User ID**: 87
- **Username**: test_hi_en
- **Language Pair**: Hindi → English (ID: 91)
- **Words in database**: 0

### Word Sets
- 20 public Hindi A1 General word sets exist (IDs 11492-11511)
- Each set contains 40-50 words
- Example: Word Set 11492 "Hindi A1: General 1" has 41 words

### Critical Finding
The `source_words_hindi` table contains **9999 total words**:
- **9966 placeholders** (99.67%)
- **33 real words** (0.33%)

**Examples of placeholder words:**
- `हाँ_9_A1` (yes_9_A1)
- `हाँ_10_A1` (yes_10_A1)
- `हाँ_11_A1` (yes_11_A1)
- `करना_918_A2` (to do_918_A2)
- `करना_919_A2` (to do_919_A2)

**All 33 real words:**
- A1 (9 words): हाँ, नहीं, मैं, तुम, वह, घर, पानी, दिन, रात
- A2 (7 words): करना, कर सकते हैं, धन्यवाद, कृपया, समय, साल, आज
- B1 (5 words): वास्तव में, आमतौर पर, शायद, स्पष्ट रूप से, बेशक
- B2 (5 words): हालांकि, फिर भी, इसके अलावा, इसलिए, यानी
- C1 (4 words): निस्संदेह, अपरिवर्तनीय, निर्विवाद, (1 more)
- C2 (4 words): अथाह, अवर्णनीय, अप्राप्य, अपरिहार्य

### Database Structure Issues
1. **Missing POS data**: Most words have `pos: null`
2. **Missing examples**: Most words have `example_hi: null`
3. **Placeholder format**: Words like `हाँ_9_A1` suggest autogenerated placeholders

### Frontend Display
The screenshot from lexybooster.com shows word sets displaying these placeholders:
- ही_501_A1 financial
- ही_502_A1 things
- ही_503_A1 working
- ही_504_A1 against

## Root Cause
The Hindi source words table was populated with placeholder data during generation/migration, and these placeholders were never replaced with actual Hindi words.

## Impact
- Users learning Hindi see meaningless placeholder text
- Cannot effectively learn Hindi vocabulary
- Affects all Hindi word sets (A1, A2, B1, B2, C1, C2 levels)

## Next Steps
1. Identify source of correct Hindi vocabulary
2. Create migration script to replace placeholders with real words
3. Ensure translations and examples are properly populated
4. Verify POS (part of speech) data is included
5. Test with user 87 account
