## Technical Considerations

### Vocabulary Reuse
- German words: Can be reused across de-ru, de-en, en-de, es-de, fr-de, it-de
- English words: Can be reused across en-ru, en-de, es-en, fr-en, it-en, ru-en
- Translation effort: Only need to translate, not recreate vocabulary lists

### Database Structure
Already supports:
- `language_pairs` table with `from_lang` and `to_lang`
- `global_word_collections` linked to language pairs
- Multi-language interface support

### Resource Estimation
- **Translation time:** ~2-3 hours per 1,000 words
- **Quality review:** ~1 hour per 1,000 words
- **Testing:** ~30 min per language pair
- **Total per complete pair (10,800 words):** ~40-50 hours

### Automation Opportunities
- Use translation APIs for initial draft
- Human review for quality
- Reuse example sentences where culturally appropriate
- Template-based generation for systematic vocabulary
